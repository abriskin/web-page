Basic Info

Web developers create websites. They're responsible for the technical aspects (performance, capacity, etc.), which are measures of how fast the website is and how much traffic it can handle. Web designers create and test a website or interface's layout, functions, and navigation for usability. They're also responsible for the look of the website or interface.
