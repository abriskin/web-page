Basic Info

Web developers create websites. They're responsible for the technical aspects (performance, capacity, etc.), which are measures of how fast the website is and how much traffic it can handle. Web designers create and test a website or interface's layout, functions, and navigation for usability. They're also responsible for the look of the website or interface.

Job Specifics

Web developers/designers usually do the following:

  • Meet with clients
  • Create and test apps, interfaces, or navigation for a site
  • Write code for the site
  • Work with team members to determine the info on the site
  • Work with team members to determine the site's layout
  • Integrate graphics, audio, and video on the site
  • Moniter site traffic
  • Create prototypes/mockups of sites or apps
  • Design and develop graphics

One of the main focuses of creating a website is making the client's vision a reality, and choosing the right applications to build the type of site that's needed. A developer decides which applications and designs best fit the site, and the designer focuses on the look and usability of those elements.

Job Variations

Some developers/designers are proficient in all aspects of a site's construction, while others specialize in a certain area. Here are four examples of specialized web designers/developers:

BACK-END DEVELOPERS

Build the foundations of the website. They create the basic framework and make sure that it functions correctly. They also establish procedure for allowing others to add new pages and meet with management to discuss major changes.

FRONT-END DEVELOPERS

Build a website's look. They create the site's layout and integrate graphics, applications, and other content.

WEB/DIGITAL INTERFACE DESIGNERS

Build the look and feel of a site or interface in regard to font, color, graphics, and layout. They're also responsible for the functionality and usability of the site/interface.

WEBMASTERS

Maintain and update websites. They make sure that sites operate correctly and also sometimes respond to user comments.
